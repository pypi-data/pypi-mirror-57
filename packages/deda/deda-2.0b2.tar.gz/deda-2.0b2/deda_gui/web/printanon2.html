Print Anonymization


1. First generate a print mask for the specific printer you want to use.

2.Load generated printer mask (.json) and the document (PDF) to generate an anonymous document. This document is saved as "documentname_masked.pdf" in the same folder as the document. Additional an settings file is saved as "calibration_masked.txt". Print the anonymous document borderless and proceed to step 3.
Generate Document

3. Scan artifacts can cause a small offset between the original and masked yellow dots. To fix this, scan the anonymized print and check the yellow dot patterns. If there is an offset - vary x and y to overlay the masking points exactly with the original yellow points. Repeat step 2 with x and y set and click Generate Document again. The dot radius describes the size of a tracking dot to be printed.
